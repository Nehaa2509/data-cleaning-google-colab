# Data Cleaning & Missing Value Handling

## Description
This project demonstrates data cleaning and preprocessing using Python and Pandas in Google Colab.

## Datasets Used
- House Prices Dataset
- Medical Appointment No Shows Dataset

## Tasks Performed
- Loaded Excel datasets using Pandas
- Identified missing and invalid values
- Applied median imputation for numerical columns
- Removed logically incorrect records
- Converted date columns to datetime format
- Encoded categorical variables
- Exported cleaned datasets

## Tools & Technologies
- Python
- Pandas
- Google Colab
- Excel

## Output Files
- House_Prices_Cleaned.xlsx
- Medical_Appointments_Cleaned.xlsx
